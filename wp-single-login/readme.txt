Using wp single login you can easily add the functionality to allow only single login per user.

== Description ==
This plugin will automatically logout the already logged in user when a user with the same login details tries to login from different browser or different computer. This plugin needs zero configuration to run. Just install it if you want single login functionality on your site.

This plugin uses wordpress Heartbit API which was implemented in version 3.6 in order to logout users in real time.

== Installation ==
* Download the plugin and extract it.
* Upload the directory '/wp-single-login/' to the 'wp-content/plugins/' directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* That\'s it you are done, you site is now only allows single login with a login details.
